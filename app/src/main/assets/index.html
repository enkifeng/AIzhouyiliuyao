⚡ 动变规则
回头生：变爻生动爻 → 大吉，有根有源
回头克：变爻克动爻 → 大凶，事态逆转
回头合：动变相合 → 合绊、阻碍
进神：寅→卯/申→酉 → 前进发展
退神：卯→寅/酉→申 → 后退衰落
反吟：动变相冲 → 反复不稳
伏吟：动变相同 → 痛苦停滞
化旺：变爻被日月生 → 力量增强
超级化旺：变爻临月建日辰 → 最强

六冲：子午/丑未/寅申/卯酉/辰戌/巳亥
👻 伏神规则
伏神：本卦缺少的六亲，从伏卦（变卦）中寻找
飞神：伏神所在爻位的地支（本爻）
飞生伏：飞神生伏神 → 伏神易出，有力
飞克伏：飞神克伏神 → 伏神难出，无力
伏克飞：伏神克飞神 → 伏神可出
伏生飞：伏神生飞神 → 泄气，无力
飞空：飞神旬空 → 伏神易出
飞破：飞神月破 → 伏神易出
📌 伏神用法
1. 本卦无某六亲 → 查伏卦对应爻位
2. 伏神为用神时 → 需看飞神生克
3. 伏神出伏条件：
  • 飞神旬空/月破
  • 日月冲飞神
  • 伏神临日月
4. 伏神评分：通常-5~-10（力量较弱）
